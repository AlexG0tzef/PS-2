Sample behavioral waveforms for design file "lclk.v"

The following waveforms show the behavior of lpm_counter megafunction for the chosen set of parameters in design "lclk.v". The design "lclk.v" is a 12 bit down counter. The counter has a synchronous set signal, that sets the counter to 1745.

Fig. 1 : Wave showing counter operation.

The output port cout will be asserted at the completion of count sequence. The ports cin and cout are used to chain multiple counters together.
